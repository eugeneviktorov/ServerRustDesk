## 1. Хостинги

### 1.1. vdsina.com

После регистрации на хостинге создаём сервер, выбираем данные пункты (на данном изображении создаём чистый сервер).

Тут вы можете как раз посмотреть стоимость сервера, и далее потом пополнить баланс в иностранной валюте с помощью различных сервисов оплаты.

<img src="./photo/vdsina/1.png" />

В случае если не планируется вручную всё настраиваться, то можно воспользоваться готовой конфигурацией:

<img src="./photo/vdsina/2.png" />

После оплаты и создание сервера ожидаем статуса - активно, после чего можно подключаться к серверу. Ожидае может быть продолжительным.

<img src="./photo/vdsina/3.png" />

После активации, и открытии сервера, можно посмотреть данные по серверу: его IP-адрес и пароль для подключения.

<img src="./photo/vdsina/4.png" />

Сервис хороший в плане серверов, стабильно работает, однако сообщения от руководителя в виде информационных сообщений не всегда самые хорошие, и могут быть очень агрессивными.

Пример: если РКН заблокировал какой-то сервис, то хостинг может написать не самое приятное информационное сообщение (всем клиентам хостинга).

Нужно быть осторожным в выборе данного сервиса.

## 2. Сервисы с оплатой

### 2.1. Плати легко

Сайт - <a href="https://pay-saas.ru">https://pay-saas.ru</a>

Платформа для финансовых услуг для граждан и компаний РФ. Основные возможности:

- Оплата зарубежных сервисов и подписок для физлиц.
- Ежемесячная оплата корпоративных зарубежных сервисов для юрлиц.
- Приём и отправка переводов (в т.ч. SWIFT/IBAN).
- Эквайринг от зарубежных клиентов с зачислением средств в РФ.
- Вывод средств за рубеж и хранение валюты на иностранных счетах.
- Приём и доставка посылок от зарубежных магазинов, не отправляющих в РФ.

Плюсы: работает стабильно, общение через менеджера.

Минусы: долгие ответы (1–2 часа на выпуск временной карты). Есть предположение что компания является посредником.

### 2.2. Wanttopay

Telergam бот - <a href="https://t.me/WantToPayBot">@WantToPayBot</a>

Мини-приложение для быстрого выпуска виртуальных карт Visa/Mastercard для международных онлайн-покупок. Баланс можно пополнять в рублях и криптовалюте. Есть карты с поддержкой Apple Pay для офлайн-оплаты за рубежом.

Плюсы: возможность выпуска собственной карты.

Минусы: практический опыт работы с сервисом отсутствует.

### 2.3. CinCin Cards

Telergam бот - <a href="https://t.me/CinCinR_bot">@CinCinR_bot</a>

Мини-приложение для быстрого выпуска виртуальных карт для международных онлайн-покупок. Баланс можно пополнять в рублях. Есть карты с поддержкой Apple Pay для офлайн-оплаты за рубежом.

Плюсы: возможность выпуска собственной карты.
